Challenge 3: Single Image Deraining

Track 3: Single Image Deraining

For access to the dataset, please refer to the dataset page of our CodaLabs competition. Alternatively, the GT-RAIN dataset is available at the following link. Please note that the validation set for this competition combines both the testing and validation set of GT-RAIN. If you do download from GT-RAIN directly and would like to submit your results during the validation phase, please combine the testing and validation sets together in order for your submission to be properly evaluated. The dataset contains a README description of the GT-RAIN training and validation set. The images that will be used for the final scoring will be released during the testing phase.

If you have any questions about this challenge track please feel free to email GTRainChallenge@gmail.com
